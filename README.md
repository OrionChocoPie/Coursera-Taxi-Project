# Taxi traffic predictor

Заключительный проект для специализации Yandex/MIPT на Coursera.

[Ссылка на веб-приложение](https://sheltered-reaches-65434.herokuapp.com)

Данный проект использует Python, с применением веб-фреймворка Dash (комбинация Plotly и Flask) и хоститься на Heroku. Еще для подготовки использовался огромный стек ml-библиотек, и для деплой сама модель была упакована и делает построения по запросам на дэшборде
